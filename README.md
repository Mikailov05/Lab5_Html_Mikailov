### Заголовки и абзацы
```html
<h1>Главный заголовок</h1>
<h2>Заголовок второго уровня</h2>
<p>Это обычный абзац текста.</p>
Форматирование текста
html
<strong>Жирный текст</strong>
<em>Курсивный текст</em>
<strong><em>Жирный курсив</em></strong>
Ссылки и изображения
html
<a href="https://github.com/Mikailov05/Lab5-Shag-15_Mikailov">Мой GitHub</a>

Списки
html

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>


<ol>
  <li>Первый пункт</li>

Формы
html
<form>
  <label for="name">Имя:</label>
  <input type="text" id="name" placeholder="Введите имя">
  
  <label for="message">Сообщение:</label>
  <textarea id="message" rows="4"></textarea>
  
  <button type="submit">Отправить</button>
</form>
Семантические теги HTML5
html
<header>Шапка сайта</header>
<nav>Навигация</nav>
<main>Основное содержание</main>
<section>Логический раздел</section>
<article>Самостоятельная статья</article>
<footer>Подвал сайта</footer>

Формы: <form>, <input>, <textarea>, <select>, <option>, <button>, <label>

Семантические: <header>, <nav>, <main>, <section>, <article>, <footer>
